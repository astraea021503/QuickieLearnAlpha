Understanding CSS and HTML Principles

HTML Styles – CSS

Cascading Style Sheet (CSS): CSS is a tool to add styles (color, fonts, layout) to HTML elements.

Types of CSS:

  • Inline CSS: CSS written directly within an HTML element’s style attribute. Example: 
    <p style="color: blue;">This is styled with inline CSS.</p>
  • Internal CSS: CSS inside <style> tags within the <head> of an HTML document. Example: 
    <style>
  p {
    color: green;
  }
</style>
  • External CSS: CSS in a separate file (e.g., styles.css) and linked to the HTML document. Example: 
    <link rel="stylesheet" href="styles.css">

CSS Properties:

  • Colors: color, background-color.
  • Fonts and Sizes: font-family, font-size.
  • Borders, Padding, and Margin: Examples of element spacing and structure.

HTML Links

Syntax: The <a> tag with href for hyperlinking. Example: 

Visit Example

  • target Attribute: Opens links in new tabs with target="_blank".
  • Image as a Link: Wrapping images within <a> tags to make them clickable.
  • Button Styled Links: Using CSS to style links as buttons for a more interactive look.

CSS Links

Styling Links: Use pseudo-classes like a:link, a:hover, a:visited, a:active.

  • Removing Underlines and Adding Color: 
    a {
    text-decoration: none;
    color: #333;
}
  • Link Buttons: CSS properties to style links like buttons.

HTML Images

Image Syntax: Basic <img> tag. Example: 

<img src="quickielearn.png" alt="Example Image" height="10">

  • src Attribute: Specifies the image path.
  • alt Attribute: Provides alternative text.
  • Width and Height: Resizing with attributes or CSS properties.
  • Floating Images: Aligning images with CSS float for page layout.
