Explain: Mastering CSS Grids - Advanced Layouts and Image Galleries

1. What is CSS Grid?

Purpose: CSS Grid is a layout system that allows for more complex and responsive designs by dividing a page into rows and columns.

Function:

  • CSS Grid provides a way to structure your web page with more control over positioning elements in both directions: horizontally and vertically.
  • It is a 2-dimensional system, which is perfect for creating responsive layouts with ease.
  • CSS Grid simplifies complex layouts that were previously difficult or required many workarounds using other methods like float or positioning.

2. Defining a Grid Container

Purpose: To create a grid, you must define a grid container, which will hold the grid items.

Function:

  • Use the display: grid; property to turn a container into a grid container.
  • Example: 
    .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}
  • You can use the grid-template-rows property to define how many rows your grid will have.
  • For more flexible layouts, you can use auto values for grid rows or columns.

3. Grid Tracks (Columns and Rows)

Purpose: You can define the size of columns and rows in a grid to control the layout more precisely.

Function:

  • The grid-template-columns property defines the columns of the grid.
  • The grid-template-rows property defines the rows of the grid.
  • Example for a 3-column layout: 
    .container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
}
  • You can also use fractional units (fr) for flexible space distribution within grid tracks.

4. Grid Gaps

Purpose: The gap property controls the space between grid items.

Function:

  • Use the gap property to define the space between rows and columns.
  • Example: 
    .container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    gap: 15px;
}
  • The gap property makes it easier to create uniform spacing without needing margins on individual grid items.

5. Creating an Image Gallery

Purpose: CSS Grid is often used to create responsive image galleries.

Function:

  • By using grid-template-columns with percentage values or auto, you can create a gallery that adapts to different screen sizes.
  • Example: 
    .gallery {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 20px;
}
  • Images will automatically adjust to fill the space based on the screen size, creating a responsive and flexible layout.

6. Grid Item Placement

Purpose: CSS Grid allows precise control over where an item is placed within the grid.

Function:

  • You can control an itemâ€™s position by using grid-column and grid-row properties.
  • Example: 
    .item {
    grid-column: 1 / 3;
    grid-row: 1 / 2;
}
  • This example places the item across the first and second columns, and the first row.
  • You can also use span to make an item span multiple columns or rows.

7. Creating Complex Layouts

Purpose: CSS Grid makes creating complex layouts more intuitive, such as two-dimensional grids, overlapping items, and asymmetrical designs.

Function:

  • For a complex layout, you can combine both grid-template-columns and grid-template-rows to create various grid structures.
  • Example: 
    .container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-rows: auto 1fr auto;
    gap: 10px;
}
  • This layout has three columns and three rows, with the middle row stretching to fill available space.

8. Aligning Items in a Grid

Purpose: CSS Grid allows you to control the alignment of grid items along both axes.

Function:

  • The align-items property aligns grid items along the block (vertical) axis.
  • The justify-items property aligns grid items along the inline (horizontal) axis.
  • Example: 
    .container {
    display: grid;
    align-items: center;
    justify-items: center;
}
  • This centers all items both vertically and horizontally within their respective grid cells.

Example: Complex Grid Layout

.container {
    display: grid;

    grid-template-rows: auto 1fr auto;
    gap: 10px;

    grid-row: 1 / 2;

    grid-column: 2 / 3;
    grid-row: 2 / 3;

    grid-column: 1 / 2;
    grid-row: 2 / 3;
