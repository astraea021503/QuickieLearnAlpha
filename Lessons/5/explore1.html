Understanding CSS Grid

See How CSS Grid Works in Practice!

Now that you have an idea of what CSS Grid can do, let's dive into a practical example! In this section, we'll break down the concept of CSS Grid and explore its powerful features to build flexible and responsive layouts.

CSS Grid Basics

CSS Grid uses a system of rows and columns. You define the grid, and then place items in it.

Defining Grid Areas

You can define specific areas within your grid layout for elements to fit inside.

Responsive Layout

CSS Grid makes it easy to create layouts that adjust for different screen sizes.

Example Code for a Basic CSS Grid Layout

<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
</div>

<style>
  .grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
  }
  .grid-item {
    background-color: #f8f9fa;
    padding: 20px;
    border: 1px solid #007bff;
    text-align: center;
  }
</style>

This simple code creates a 3-column layout where each item is equally spaced within the grid container. Play around with it to see how it responds when you resize the screen!
