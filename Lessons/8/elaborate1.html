Elaborate: Build Your Own Webpage

Use the following instructions to build a simple webpage:

  • Create a heading with <h1>.
  • Add a paragraph using <p>.
  • Style the paragraph with a background color using the style attribute.

Example code:

<h1>Welcome to My Webpage</h1>
<p style="background-color: yellow;">This is a styled paragraph.</p>
