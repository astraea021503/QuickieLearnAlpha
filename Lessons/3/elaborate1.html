Elaborate Phase: Advanced HTML and CSS

Putting It All Together

Now that you have explored the basics of backgrounds, lists, and tables, it's time to apply these concepts to build something functional and visually appealing. This phase will challenge you to integrate these elements into a cohesive design.

Activity: Building a Portfolio Section

Your task is to create a small portfolio webpage using the following requirements:

  • Set a fixed background image for the page with custom positioning and size.
  • Create an unordered list to display your skills. Use custom list styles to make it visually appealing.
  • Build a table to showcase at least three projects. Include columns for project name, description, and status.

Additional Instructions

Incorporate the following CSS properties for a polished look:

  • Box Shadows: Use box shadows to highlight the table and list sections.
  • Gradients: Add gradient backgrounds for section headers.
  • Hover Effects: Style table rows or list items to change appearance on hover.
