Explore Backgrounds, Lists, and Tables

Unleashing the Potential of HTML and CSS

In this section, you'll explore how HTML and CSS work together to create visually appealing and functional web pages. This phase includes hands-on practice and detailed explanations of key concepts.

Backgrounds

Backgrounds are essential for enhancing your website's visual appeal. CSS offers extensive control over backgrounds:

  • background-color: Use a hexadecimal value, RGB, or color name to add color. Example: background-color: #f0f0f0;.
  • background-image: Set an image or gradient as the background. Example: background-image: linear-gradient(to right, #007bff, #0056b3);.
  • background-attachment: Control if the background scrolls or is fixed. Example: background-attachment: fixed;.
  • background-position: Set the starting position of the background. Example: background-position: center;.

Practice creating backgrounds for headers, sections, and full pages.

Lists

Lists help organize information and improve readability:

  • Ordered Lists: Used for sequences or rankings. Example:
    1. HTML Basics
    2. CSS Styling
    3. JavaScript Fundamentals
  • Unordered Lists: Suitable for items without a specific order. Customize with list-style-type (e.g., circle, disc, square).
  • Nested Lists: Create sub-lists for detailed information:
    • Main Topic
      • Sub-topic 1
      • Sub-topic 2

Tables

Tables are indispensable for presenting data. Advanced CSS properties allow you to style them effectively:

  • border-spacing: Adjust spacing between cells.
  • nth-child: Style alternating rows or columns.
  • caption-side: Place captions at the top or bottom of the table.
