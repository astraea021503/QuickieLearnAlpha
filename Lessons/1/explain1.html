HTML Fundamentals

Explain: Understanding HTML Tags and Attributes

1. <!DOCTYPE html>

Purpose: This declaration defines the document type and version of HTML being used. It informs the web browser about the HTML version to expect, helping it to render the page correctly.

Function:

  • The <!DOCTYPE> declaration must be the very first thing in an HTML document, before the <html> tag.
  • In HTML5, the declaration is simply <!DOCTYPE html>, indicating that the document is using HTML5.

2. <html>

Purpose: This element represents the root of an HTML document. It is the container for all other HTML elements (except the <!DOCTYPE> declaration).

Function:

  • The <html> tag indicates that everything within it is an HTML document.
  • It can also have attributes like lang, which specifies the language of the document (e.g., <html lang="en"> for English).

3. <head>

Purpose: This element contains meta-information about the HTML document, such as its title, character set, styles, scripts, and other metadata that are not displayed directly on the web page.

Function:

  • Inside the <head>, you can include:
    • <meta> tags for defining character set (<meta charset="UTF-8">), viewport settings for responsive design, etc.
    • <link> tags to link external CSS stylesheets.
    • <script> tags to link or include JavaScript files.
    • <title> tag to set the title of the document, which appears on the browser tab.

4. <title>

Purpose: The <title> element defines the title of the HTML document.

Function:

  • The content of the <title> tag is displayed in the browserâ€™s title bar or tab and is also used by search engines as the title of the webpage in search results.
  • A well-crafted title is important for SEO (Search Engine Optimization) and should accurately reflect the content of the page.

5. <body>

Purpose: This element contains all the content of the HTML document that will be displayed to the user, such as text, images, links, and other media.

Function:

  • The <body> tag encompasses everything that is visible on the web page, including headings, paragraphs, lists, tables, images, and more.
  • It can also include event-handling attributes (like onclick) for interactivity and dynamic content, as well as inline styles or references to external stylesheets for styling the content.

Example Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Example HTML Document</title>
</head>
<body>
    <h1>Welcome to My Web Page</h1>
    <p>This is a sample paragraph on my web page.</p>
</body>
</html>
